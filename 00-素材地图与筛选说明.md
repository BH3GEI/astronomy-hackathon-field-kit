## Agent 入口

这个仓库优先给 agent 调用，不是给人顺序阅读。

1. 先加载 `skills/astronomy-background-brainstorm`。
2. 再读这份素材地图建立索引。
3. 做 brainstorming 时，按 skill 的 evidence gate 输出项目种子：每个方向都要指向本仓库里的具体素材路径。
4. 遇到当前天象、天气、可见性、软件版本、API、许可等会变化的信息，必须联网或用实时数据重新验证。

## 素材地图

1. `01-共同语境与背景课件`
   - 代表文件：`浅谈天球坐标系与时间历法.pptx`、`天球坐标系与历法简介.pptx`、`太阳系概观 杨佳悦.pptx`、`天文学习班.pptx`、`恒星.pptx`、`银河系.pptx`。
   - 用途：给团队提供共同语境：坐标、历法、望远镜、太阳系、恒星、银河系，更像项目讨论时随手翻的背景层。

2. `02-观测活动与目标`
   - 重点：`野外观测指南/吉大天协野外观测目标.xlsx`、`野外观测指南/观测目标(1).doc`、`补充参考/观测目标修改后1.docx`、`补充参考/【观测部】波特尔星空等级分类法.docx`。
   - 用途：可以直接转成“今晚看什么”“观测地质量评估”“活动注意事项”“目标推荐器”等项目。
   - 注意：流星雨日历和 46P 彗星表是历史样例，不适合直接当 2026 年观测日程。

3. `03-望远镜赤道仪与拍摄前期`
   - 代表文件：`拍星准备步骤说明.doc`、`赤道仪入门手册 (1)(1).doc`、`【观测部】漂移法对极轴详细图解.doc`、`赤道仪系统的导星原理.pdf`、`摄影极限星等计算.pdf`。
   - 用途：适合做“观测准备 checklist”“对极轴辅助”“曝光/极限星等计算器”“彗星/ISS 追踪解释器”。

4. `04-天文摄影后期与实验数据`
   - 后期流程：NEPTUNE 深空后期四篇，覆盖校正、拉伸、降噪、星点分离。
   - 图像理解：游戏猫三篇，覆盖频率、颜色、预处理/信噪比。
   - 工具材料：`DSS常规使用方法.doc`、`PixInsight LE 学习笔记(v1.1) By Steven Ni.pdf`、`HG版深空处理教程.pdf`。
   - 参考大部头：`天文摄影向导...pdf` 和 `Scientific Astrophotography.pdf`。
   - `天蝎座调色盘RAW文件(练手用）.zip` 是可直接进图像 pipeline 的原始素材。

5. `05-样例照片与素材`
   - 精选了月球、木星、日凌、星轨、流星、银河、星云、星团、星系和彗星照片。
   - `原始RAW与素材/银河素材.zip` 可用于图像处理、降噪、叠加、识别类 demo。
   - `照片候选缩略图墙.jpg` 是我这轮筛图时的缩略预览。

6. `06-题库与科普问答`
   - `每日天文题` 是高价值题库：可做科普问答、知识图谱、答题 Bot、活动热身赛。
   - 这里没有搬 `每日天文题 (1)`，因为它和 `每日天文题` 大量重复，而且后者更完整。

## 暂不纳入主线的东西

- `安装包`：大多是旧 Android APK、Windows 绿色版、破解补丁或旧软件包；复用价值低，安全和版权风险高。
- `压缩包`：有 MaxImDL keygen、旧 DeepSkyStacker、旧插件等，除非专门复古复现实验，否则不建议作为黑客松基础资料。
- `科幻`：适合放松，不是这次天文黑客松的技术素材。
- `天文相册` 整包：2GB，内容很多但重复/展示向/尺寸过大，所以只搬了代表样例。
- `天文观测摄影知识/Photoshop小技巧之如何制作星轨.mp4`：114MB，内容可能有用但太大；如果之后要专攻星轨后期，再从 Downloads 单独取。

## 项目种子

- 天文问答/闯关：从 `每日天文题` 抽题，按主题分级。
- 观测目标推荐：用目标表、星等、观测时间、设备要求做推荐器。
- 观测准备控制台：把野外指南和拍星步骤变成 checklist。
- 图像处理 demo：用 RAW/银河素材做叠加、降噪、拉伸、星点分离可视化。
- 观测地评估：用波特尔等级、极限星等、照片样例做光污染/观测质量解释器。
