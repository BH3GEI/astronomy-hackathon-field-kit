# Astronomy Background Lenses

Use these lenses to turn the field kit into hackathon ideas. Combine at least two lenses per idea so the result is not just "make a page about astronomy."

## Lenses

- **Sky as coordinate system**: celestial sphere, RA/Dec, altitude/azimuth, sidereal time, calendars, visibility windows, "where is it tonight" interfaces.
- **Sky as observing mission**: target selection, Bortle class, moon phase, weather, safety checklist, equipment readiness, group activity planning.
- **Telescope as workflow**: mount setup, polar alignment, guiding, exposure limits, tracking errors, field-of-view calculators, step-by-step assistants.
- **Astrophotography as data pipeline**: calibration, stacking, registration, stretching, denoising, star separation, color mapping, before/after explainers.
- **Images as inspectable artifacts**: object recognition, annotation, quality scoring, metadata extraction, thumbnail walls, dataset curation.
- **Questions as interaction layer**: quiz bot, public outreach game, adaptive explanations, knowledge graph, "ask a senior club member" simulator.
- **Historical club files as culture**: activity templates, judging themes, volunteer workflow, observational storytelling, archive-to-event conversion.

## Project Patterns

- **Planner**: choose an observing target or activity based on constraints.
- **Explainer**: convert a concept into an interactive visual or dialogue.
- **Assistant**: guide a participant through a workflow with checklists and feedback.
- **Analyzer**: process files, images, or tables and return measurements or rankings.
- **Game**: turn question banks and images into a fast event interaction.
- **Dataset builder**: normalize messy historical files into structured JSON/CSV for later agents.
- **Judge helper**: map submissions to themes, evidence, novelty, and demo quality.

## Evidence Gate

Every idea should cite at least one local path. Stronger ideas cite one concept source plus one usable artifact, for example:

- `01-共同语境与背景课件` + `06-题库与科普问答`
- `02-观测活动与目标` + `03-望远镜赤道仪与拍摄前期`
- `04-天文摄影后期与实验数据` + `05-样例照片与素材`

## Verification Gate

Mark an idea as needing live verification when it depends on current or changing facts:

- tonight's visible objects, weather, moon phase, meteor showers, comet positions, or satellite passes
- current software availability, package versions, model capabilities, API pricing, or licensing
- safety, venue, travel, light-pollution, or equipment availability constraints
